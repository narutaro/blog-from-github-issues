    AWS

    CLI - Named Profile

    You can name profiles and switch them around.

    Create a profile

    aws configure --profile <profile_name>

    The keys will be stored in ~/.aws/credentials, region goes to ~/.aws/config

    Use a profile

    export AWS_PROFILE=<profile_name>

    Related commands and configuration files:

    • aws configure list
    • aws sts get-caller-identity

    Cloudwatch logs

    You can tail -f for the Amazon CloudWatch logs.

    Lambda logs

    aws logs tail --follow /aws/lambda/<function_name>

    You don't have to put full log group name. Just hit tab after /aws to complement the rest.

    IoT Core logs

    aws logs tail --follow AWSIotLogsV2
